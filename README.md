

# Setup Workspace
1. Clone the repositiory
```bash
git clone https://github.com/chingu-voyages/V57-tier3-team-33.git
 ```
 2. installs dependencies from the root 
 ```bash
 npm install 
 ```




## Development Commands

From the **root folder**, you can run:

```bash
npm run dev:server   
npm run dev:client   
npm run dev          # Start both
```
